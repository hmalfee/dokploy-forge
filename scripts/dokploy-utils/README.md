# Dokploy Utils

The scripts in this directory were literally cloned directly from the official [Dokploy/templates](https://github.com/Dokploy/templates) repository on the `canary` branch.

They are used to natively validate templates and `docker-compose.yml` files using the exact same logic as Dokploy itself.

### Source Files

- [`validate-template.ts`](https://github.com/Dokploy/templates/blob/canary/build-scripts/validate-template.ts)
- [`validate-docker-compose.ts`](https://github.com/Dokploy/templates/blob/canary/build-scripts/validate-docker-compose.ts)
- [`helpers.ts`](https://github.com/Dokploy/templates/blob/canary/build-scripts/helpers.ts)
- [`types.ts`](https://github.com/Dokploy/templates/blob/canary/build-scripts/types.ts)
